# clamshellctl Homebrew Tap

Install the CLI:

```sh
brew tap dotcom07/clamshellctl
brew install clamshellctl
```

Install the menu bar app:

```sh
brew tap dotcom07/clamshellctl
brew install --cask clamshellctl-app
```

Upgrade later:

```sh
brew update
brew upgrade clamshellctl
brew upgrade --cask clamshellctl-app
```
